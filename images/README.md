# Thư mục ảnh sản phẩm

Khi có ảnh sản phẩm thật, lưu vào đây với tên file sau:

- `sl207.jpg` — Ổ điện du lịch SL207
- `ol212.jpg` — Ổ điện du lịch OL212
- `jp395.jpg` — Sạc dự phòng JP395
- `rc502.jpg` — Sạc dây rút RC502

Định dạng đề xuất:
- **JPG** hoặc **PNG** (JPG tốt hơn cho ảnh có chi tiết, PNG tốt cho ảnh nền trong suốt)
- **Tỉ lệ vuông** (1:1) — ví dụ 1200×1200px
- **Kích thước file**: dưới 500KB mỗi ảnh (dùng [tinypng.com](https://tinypng.com) để nén)

Sau khi thêm ảnh, mở README.md ở thư mục cha để xem hướng dẫn sửa HTML/CSS cho ảnh hiển thị.
